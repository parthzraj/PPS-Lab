# 🧠 Programming for Problem Solving (PPS) Lab

Welcome to the **Programming for Problem Solving (PPS) Lab Repository** 👨‍💻

## 📚 About the Course

Programming for Problem Solving (PPS) is designed to build strong programming fundamentals and logical thinking skills.  
The course focuses on writing structured programs and solving real-world computational problems using the C programming language.

- **Programming Language:** C

## 🚀 How to Run the Programs

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/PPS-Lab.git
```

### Step 2: Navigate to the Folder

```bash
cd PPS-Lab
```

### Step 3: Compile the Program

```bash
gcc program.c -o program
```

### Step 4: Run the Program

```bash
./program
```
